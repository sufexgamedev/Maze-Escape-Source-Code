# Maze Escape (Unity 2D)
Hi, My name is Abu Sufiyan Arshad and this is my second game project built using Unity and C#.
<br>
It is a Top-Down puzzle game where the player navigates through a maze, collects coins, finds the key, and unlocks the door to escape and complete the level.

# 🎮 Features
- Top-Down maze gameplay
- Coin collection system
- Key and door unlocking mechanic
- 9 levels
- Smooth player movement
- Simple and clean UI

# 🛠️ Built With
- Unity (2D)
- C#

# 🚀 Purpose
This project was created as my second step into Unity Game Engine to improve and learn:

- Level design
- Game logic and mechanics
- Player controls and movement
- Working with multiple levels
- Improving problem-solving skills in game programming

# How to Play
- Use **WASD** or **Arrow Keys** to move the player
- Collect all coins in the maze
- Find the key to unlock the door
- Reach the door to escape and complete the level
